# Smart Import - Quick Reference\n\n## One-Minute Overview\n\n**Smart Import** lets photographers paste raw questionnaire data (any format) and the system intelligently extracts and organizes everything.\n\n### What It Does\n✅ Accepts any format (bullet points, tables, emails, Google Forms, PDF text, messy notes)\n✅ Intelligently parses names and relationships\n✅ Automatically categorizes attendants, family, and portrait requests\n✅ Shows a preview before import\n✅ Fully editable after import\n\n### Where to Find It\n- Dashboard → Smart Import button\n- Photographer Workspace → New Shotlist → Smart Import tab\n\n---\n\n## Supported Input Formats\n\n| Format | Example |\n|--------|----------|\n| **Bullet Points** | • Katie - Maid of Honor |\n| **Dashes** | - Katie - Maid of Honor |\n| **Parentheses** | Katie Soost (Maid of Honor) |\n| **Tables** | Name \\| Role \\| Katie \\| MOH |\n| **Paragraphs** | Katie is the Maid of Honor |\n| **Emails** | Copy entire email thread |\n| **Google Forms** | Copy form response |\n| **PDF Text** | Copy & paste from PDF |\n| **Mixed** | Combine any of the above |\n\n---\n\n## How to Use (4 Steps)\n\n### 1️⃣ Copy the Data\n```\n- Open couple's questionnaire/email/form\n- Select all text (Ctrl+A)\n- Copy (Ctrl+C)\n```\n\n### 2️⃣ Open Smart Import\n```\n- Click \"Smart Import\" button\n- See the large text area\n```\n\n### 3️⃣ Paste the Data\n```\n- Click in text area\n- Paste (Ctrl+V)\n- Don't worry about formatting!\n```\n\n### 4️⃣ Click \"Smart Import\"\n```\n- System processes data\n- See preview of extracted info\n- Review and confirm\n```\n\n---\n\n## What Gets Extracted\n\n### ✓ Always Extracted\n- Attendant names and roles\n- Family member names and relationships\n- Portrait requests\n- Special groupings\n\n### ? Might Need Manual Addition\n- Wedding date\n- Ceremony location\n- Reception location\n\n### ✗ Safely Ignored\n- Phone numbers\n- Addresses\n- Unrelated info\n\n---\n\n## Recognized Relationship Types\n\n### Attendants\n- Maid of Honor / Co-Maid of Honor\n- Best Man\n- Bridesmaids / Groomsmen\n- Attendants\n- And more variants\n\n### Family\n- Mother / Father\n- Sister / Brother\n- Cousin\n- Aunt / Uncle\n- Grandparents\n- Step-Parents\n- In-Laws\n- Co-Parents\n- Adoptive Parents\n- And 50+ more\n\n### Section Keywords\n- **Attendants:** Bridesmaids, Groomsmen, Attendants, Maids, Grooms\n- **Family:** Family, Parents, Father, Mother, Siblings\n- **Portraits:** Portraits, Requested, Portrait Requests\n- **Groupings:** Groupings, Special Groupings, Custom Groupings\n\n---\n\n## Example Inputs That Work\n\n### Example 1: Simple List\n```\nBridesmaids\n- Katie Soost - Maid of Honor\n- Rachel Leigon - Bridesmaid\n\nGroom's Family\n- Robert Chen - Father\n- Patricia Chen - Mother\n```\n✓ Works perfectly!\n\n### Example 2: Messy Email\n```\nHey! Here's our wedding party:\n\nKatie is the MOH, Rachel and Emily are bridesmaids.\nGroom's groomsmen are Mark (best man), David, and Tom.\n\nFamily stuff:\nBride's parents are Steve and Diana\nGroom's parents are Robert and Patricia\n\nWe want pics of:\n- Just us\n- With parents\n- Full party\n```\n✓ Parses everything!\n\n### Example 3: Google Form\n```\nBride's Name: Sarah\nGroom's Name: James\nBridesmaids: Katie (MOH), Rachel\nGroom's Groomsmen: Mark (BM), David\nBride's Parents: Steve, Diana\nGroom's Parents: Robert, Patricia\nPortrait Requests: Couple shot, family photos\n```\n✓ Extracts all data!\n\n### Example 4: Messy Notes\n```\nWedding Day Details\n\nAttendants:\n• Katie (Maid of Honor) - Sarah's best friend\n• Rachel - bridesmaid\n\nFamily:\nSteve - Dad\nDiana - Mom\nRobert (James's dad)\nPatricia (James's mom)\n\nPhotos:\n- Couple shot\n- With families\n- Everyone together\n```\n✓ All organized!\n\n---\n\n## What Happens After Import\n\n1. **Review Preview**\n   - See exactly what was extracted\n   - All people organized by category\n   - All relationships labeled\n\n2. **Click \"Import Data\"**\n   - Data populates photographer workspace\n   - Ready for shot list generation\n   - Everything is fully editable\n\n3. **Edit If Needed**\n   - Click on any person to edit\n   - Change names, relationships, etc.\n   - Add or delete entries\n\n4. **Generate Shot List**\n   - Auto-generation creates professional groupings\n   - Organizes into 8 categories\n   - Ready for photographer to refine\n\n---\n\n## Troubleshooting\n\n| Problem | Solution |\n|---------|----------|\n| Text won't paste | Check clipboard, try copying again |\n| Nothing extracted | Make sure you included names and roles |\n| Wrong relationships | You can edit after import |\n| Missing family members | Paste their section again and import |\n| Parsing seems wrong | Preview shows exactly what was found |\n\n---\n\n## Pro Tips\n\n💡 **Tip 1:** Messy input is fine - the system handles it!\n\n💡 **Tip 2:** \"Name - Relationship\" format works best\n\n💡 **Tip 3:** Include clear section headers (Bridesmaids, Family, etc.)\n\n💡 **Tip 4:** Edit after import if anything needs adjustment\n\n💡 **Tip 5:** Works with any language and character set\n\n---\n\n## Supported Wedding Types\n\n✅ Traditional (Bride & Groom)\n✅ Same-Sex (Two Brides, Two Grooms)\n✅ Non-Binary Partners\n✅ Multiple Parents (Co-Parents, Step, Adoptive)\n✅ Blended Families\n✅ Any Configuration\n\nThe system recognizes and honors all family structures equally.\n\n---\n\n## Common Input Patterns\n\n### Pattern 1: \"Name - Role\"\n```\nKatie Soost - Maid of Honor\n```\n\n### Pattern 2: \"Name (Role)\"\n```\nKatie Soost (Maid of Honor)\n```\n\n### Pattern 3: \"Name role\"\n```\nKatie Soost Maid of Honor\n```\n\n### Pattern 4: Table rows\n```\nKatie | Soost | Maid of Honor\n```\n\n### Pattern 5: No role (OK)\n```\nKatie Soost\n```\nWill be assigned as \"Guest\" and can be edited.\n\n---\n\n## When to Use Smart Import\n\n✅ Use Smart Import when:\n- Couple sent you a questionnaire\n- You have their email response\n- You have a Google Form response\n- You want to save time\n- Input is messy/unstructured\n\n❌ Don't use when:\n- You need to collect info yourself\n- Couple prefers step-by-step entry\n- You're meeting in person\n\n---\n\n## Quick Checklist\n\n- [ ] Copy couple's questionnaire data\n- [ ] Open Smart Import\n- [ ] Paste the data\n- [ ] Click \"Smart Import\" button\n- [ ] Review the preview\n- [ ] Click \"Import Data\"\n- [ ] Edit any details if needed\n- [ ] Generate shot list\n- [ ] Done! ✓\n\n---\n\n## Need More Help?\n\n- **Full guide:** See SMART_IMPORT_GUIDE.md\n- **Examples:** See SMART_IMPORT_GUIDE.md section \"Examples by Wedding Type\"\n- **Troubleshooting:** See SMART_IMPORT_GUIDE.md section \"Troubleshooting\"\n\n---\n\n**Smart Import: Because photographers should focus on photography, not data entry.**\n

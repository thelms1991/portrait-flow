# Smart Import Feature - Complete Documentation\n\n## 📋 Overview\n\nThe **Smart Import** feature is a game-changer for photographers. Instead of manually entering or asking couples to fill out a structured form, photographers can simply paste whatever questionnaire data they received—in any format—and the system intelligently parses and organizes it.\n\n### The Problem It Solves\n\n❌ **Before Smart Import:**\n- Couples send messy questionnaires in various formats\n- Photographers must manually re-enter the data\n- Data entry is time-consuming and error-prone\n- Different formats require different handling\n- Photographer frustration → Photo shoot delays\n\n✅ **With Smart Import:**\n- Photographers paste the data exactly as received\n- System automatically parses and organizes\n- No manual data entry needed\n- Works with any format (email, PDF, Google Forms, bullet points, tables, etc.)\n- Save hours per wedding\n\n---\n\n## 🎯 Key Features\n\n### 1. **Universal Input Support**\nAccepts any format without requiring reformatting:\n- ✅ Bullet points\n- ✅ Dashes/hyphens\n- ✅ Parentheses\n- ✅ Tables\n- ✅ Paragraphs\n- ✅ Copy-pasted emails\n- ✅ Google Form responses\n- ✅ PDF text\n- ✅ Messy notes\n- ✅ Mixed formats\n\n### 2. **Intelligent Parsing**\nRecognizes and extracts:\n- 🎭 Attendant names and roles (Maid of Honor, Best Man, etc.)\n- 👨‍👩‍👧 Family member names and relationships (50+ types)\n- 📸 Portrait request descriptions\n- ✨ Special grouping requests\n- 🏷️ Custom relationship labels\n\n### 3. **Smart Categorization**\nAutomatically organizes data into:\n- Side 1 Attendants (Bride's side / Partner 1's side)\n- Side 2 Attendants (Groom's side / Partner 2's side)\n- Partner 1 Family\n- Partner 2 Family\n- Reception Portrait Requests\n- Special Groupings\n\n### 4. **Preview Before Import**\nShowcase extracted data:\n- See exactly what was parsed\n- Review all people and relationships\n- Verify correct categorization\n- Option to edit before final import\n\n### 5. **Full Editability**\nAfter import, everything is editable:\n- ✏️ Edit person names\n- ✏️ Edit relationships\n- ➕ Add missing people\n- ➖ Delete incorrect entries\n- 🔄 Move people between categories\n\n### 6. **Inclusive by Design**\nSupports all wedding types:\n- 👰 Traditional (Bride & Groom)\n- 👰👰 Same-sex (Two Brides)\n- 🤵🤵 Same-sex (Two Grooms)\n- 👥 Non-binary partners\n- 👨‍👩‍👧‍👦 Complex family structures\n- ♾️ Custom configurations\n\n---\n\n## 🛠️ Technical Implementation\n\n###
